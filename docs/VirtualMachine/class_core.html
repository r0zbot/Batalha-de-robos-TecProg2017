Detailed Description

This class controls the entire system and its properties.

The Core class is in charge of controlling the game system during the execution of its functions. This class manages the game cycle, which includes three main phases: the Load phase, when every Entity in the game is initialized. The Update phase, when the entities in the current game have their logic updated. The Render phase, when every Entity is rendered in the main screen.
